# create-furin

Official scaffolder for [Furin](https://github.com/Teyik0/furin) — the React meta-framework on Elysia + Bun.

## Usage

```bash
bun create furin my-app
bun create furin my-app --template full
bunx @teyik0/create-furin my-app
```

## Templates

| Template | Description |
|---|---|
| `simple` | Tailwind CSS + one API route |
| `full` | shadcn/ui + Tailwind CSS + API routes |

## Options

```
--template <simple|full>   Template to use (default: prompted)
--yes                      Skip confirmation prompts
--no-install               Skip bun install
--help                     Show this help
--version                  Show version
```
