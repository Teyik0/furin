# API Routes

Furin runs on [Elysia](https://elysiajs.com), so your API lives in the same process as your frontend. No separate server, no proxy.

## Defining API Routes

Create a regular Elysia group and `.use()` it in your server:

```ts
// src/api/index.ts
import { Elysia, t } from "elysia"

export const api = new Elysia({ prefix: "/api" })
  .get("/posts", async () => {
    const posts = await db.getAllPosts()
    return posts
  })
  .post("/posts", async ({ body }) => {
    const post = await db.createPost(body)
    return post
  }, {
    body: t.Object({
      title: t.String(),
      content: t.String(),
    }),
  })
```

```ts
// src/server.ts
import { furin } from "@teyik0/furin"
import { Elysia } from "elysia"
import { api } from "./api/index.ts"

const app = new Elysia()
  .use(api)
  .use(await furin({ pagesDir: "./src/pages" }))
  .listen(3000)

export type App = typeof app
```

## End-to-End Type Safety

Export `typeof app` and use it on the client with [Eden Treaty](https://elysiajs.com/eden/treaty/overview.html):

```ts
// src/client.ts
import { treaty } from "@elysiajs/eden"
import type { App } from "./server.ts"

export const api = treaty<App>("localhost:3000")
```

```tsx
// src/pages/blog/index.tsx
import { api } from "@/client"

export default route.page({
  loader: async () => {
    const { data } = await api.api.posts.get()
    return { posts: data ?? [] }
  },
  component: ({ posts }) => <PostList posts={posts} />,
})
```

Your API response types flow through automatically — no code generation, no manual typing.

## Middleware

Use Elysia's lifecycle hooks for authentication, logging, and more:

```ts
// src/api/index.ts
import { Elysia } from "elysia"
import { jwt } from "@elysiajs/jwt"

export const api = new Elysia({ prefix: "/api" })
  .use(jwt({ name: "jwt", secret: process.env.JWT_SECRET! }))
  .guard(
    { beforeHandle: async ({ jwt, cookie, set }) => {
        const user = await jwt.verify(cookie.auth.value)
        if (!user) { set.status = 401; return "Unauthorized" }
      }
    },
    (app) => app
      .get("/me", ({ cookie, jwt }) => jwt.verify(cookie.auth.value))
  )
```

## WebSockets

Elysia's WebSocket support works as expected:

```ts
// src/api/index.ts
export const api = new Elysia({ prefix: "/api" })
  .ws("/chat", {
    message(ws, message) {
      ws.publish("chat", message)
    },
  })
```
