# Getting Started

Furin is a React meta-framework built on top of [Elysia](https://elysiajs.com) and [Bun](https://bun.sh). It provides file-based routing, multiple rendering modes (SSR, SSG, ISR), and full TypeScript inference — all in a single process.

## Installation

```bash
bun create furin@latest my-app
cd my-app
bun install
```

## Project Structure

```
my-app/
├── src/
│   ├── pages/          # File-based routes
│   │   ├── root.tsx    # Root layout
│   │   └── index.tsx   # Home page (/)
│   └── server.ts       # Elysia server entry
├── furin.config.ts     # Build configuration
└── bunfig.toml         # Bun configuration
```

## Your First Server

```ts
// src/server.ts
import { Elysia } from "elysia"
import { furin } from "@teyik0/furin"

const app = new Elysia()
  .use(await furin({ pagesDir: "./src/pages" }))
  .listen(3000)

export type App = typeof app
```

## Your First Page

```tsx
// src/pages/index.tsx
import { route } from "./root"

export default route.page({
  component: () => <h1>Hello Furin!</h1>,
})
```

## Root Layout

Every app needs a root layout that wraps all pages:

```tsx
// src/pages/root.tsx
import { createRoute } from "@teyik0/furin/client"

export const route = createRoute({
  layout: ({ children }) => (
    <html lang="en">
      <body>{children}</body>
    </html>
  ),
})
```

## Development

```bash
bun run dev
```

Your app runs at `http://localhost:3000` with hot module replacement enabled.
