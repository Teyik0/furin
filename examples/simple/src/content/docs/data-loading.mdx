# Data Loading

Furin uses **loaders** to fetch data on the server before rendering. Loaders are fully type-safe — the return value is automatically inferred as component props.

## Basic Loader

```tsx
// src/pages/blog/[slug].tsx
export default route.page({
  loader: async ({ params }) => {
    const post = await db.getPost(params.slug)
    return { post }
  },
  component: ({ post }) => (
    <article>
      <h1>{post.title}</h1>
      <p>{post.body}</p>
    </article>
  ),
})
```

The `params` object is fully typed based on your file name. `[slug].tsx` gives you `params.slug: string`.

## Loader Context

Loaders receive a context object with:

| Property | Type | Description |
|---|---|---|
| `params` | `Record<string, string>` | URL path parameters |
| `query` | `Record<string, string>` | Validated query string values |
| `request` | `Request` | The raw Bun `Request` object |

```tsx
// src/pages/blog/index.tsx
export default route.page({
  loader: async ({ query, request }) => {
    const page = Number(query.page ?? 1)
    const posts = await db.getPosts({ page })
    return { posts, page }
  },
  component: ({ posts, page }) => (
    <div>
      {posts.map(p => <PostCard key={p.id} post={p} />)}
    </div>
  ),
})
```

## Query Schema Validation

Declare a `query` schema to get automatic validation and type inference:

```tsx
// src/pages/blog/index.tsx
import { t } from "elysia"

export default route.page({
  query: t.Object({
    page: t.Optional(t.Number({ default: 1 })),
    tag:  t.Optional(t.String()),
  }),
  loader: async ({ query }) => {
    // query.page is number, query.tag is string | undefined
    const posts = await db.getPosts(query)
    return { posts }
  },
  component: ({ posts }) => <PostList posts={posts} />,
})
```

## Returning Responses

Return a `Response` directly to redirect or send a custom status:

```tsx
// src/pages/dashboard.tsx
export default route.page({
  loader: async ({ request }) => {
    const user = await getSession(request)
    if (!user) return Response.redirect("/login", 302)
    return { user }
  },
  component: ({ user }) => <Dashboard user={user} />,
})
```
